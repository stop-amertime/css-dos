DOS SHELL - a plain DOS prompt, running in CSS.

This floppy carries nine classic utilities from the FreeDOS
project (freedos.org). Things to try:

  DIR              list this floppy
  TYPE README.TXT  print this file
  MORE README.TXT  ...one page at a time
  TREE             draw the directory tree
  MEM              how much memory does a CSS file have?
  EDIT README.TXT  full-screen editor (Alt for menus)
  EDLIN README.TXT the 1980 way (E to quit)
  DEBUG            poke the machine itself (Q to quit)
  FIND "CSS" README.TXT
  SORT README.TXT

The disk is read-only for now: editors open files fine but
cannot save. The CPU, the memory, the video card and the
floppy you are reading right now are all CSS custom
properties being recomputed every tick.
